# Online Banking System (Java + JDBC + MySQL)

A console-based banking system built using Java and MySQL.

## Features
- Create account
- Deposit money
- Withdraw money
- Check balance
- Transaction history

## Technologies Used
- Java
- JDBC
- MySQL
- Git & GitHub

## How to Run
1. Compile:
   javac -cp ".;mysql-connector-j-9.6.0.jar" BankApp.java

2. Run:
   java -cp ".;mysql-connector-j-9.6.0.jar" BankApp
